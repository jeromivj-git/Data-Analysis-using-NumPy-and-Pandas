# 📊 Data Analysis using NumPy & Pandas

## 📌 Overview
This project demonstrates fundamental data analysis concepts using NumPy and Pandas. It covers array operations, data manipulation, and basic data exploration techniques through practical examples.

---

## 🔹 Topics Covered

### 🧮 NumPy
- 1D & 2D Array Creation
- Array Properties (shape, dtype, size)
- Mathematical Operations
- Temperature Conversion (Celsius → Fahrenheit)
- Slicing & Indexing

### 🐼 Pandas Series
- Creating Series with custom index
- Indexing (loc, iloc)
- Filtering using conditions
- Data manipulation (update, delete)
- Basic calculations (CGPA)

### 📊 Pandas DataFrame
- Creating DataFrame
- Data Exploration (`head()`, `tail()`, `info()`)
- Column Selection & Filtering
- GroupBy operations
- Handling missing data
- Data manipulation (add, update, delete columns/rows)
